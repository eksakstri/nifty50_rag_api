# Nifty50 RAG API

A Retrieval-Augmented Generation (RAG) based API for answering queries related to Nifty50 companies, market data, and financial insights. The system combines document retrieval, embeddings, and Large Language Models (LLMs) to provide context-aware responses.

## Features

- 🔍 Semantic search using vector embeddings
- 📚 Retrieval-Augmented Generation (RAG) pipeline
- 🤖 LLM-based response generation
- 📊 Nifty50 company and market data analysis
- ⚡ FastAPI-based REST API
- 🗂️ FAISS vector database for efficient similarity search

## Tech Stack

- Python
- FastAPI
- LangChain / LangGraph
- Sentence Transformers
- FAISS
- LLM API (ChatGroq/OpenAI compatible)
- Pandas & NumPy
